## 🛠️ Installation Steps
###  npm.
  1. open Terminal in VScode or atom and clone

```bash
  git clone https://github.com/pratyush2410/HackNITP-3.0-H4CK3RS-
```

  2. Install packages

```bash
  npm install
```
  3. Run the app

```bash
  node server.js
```

🌟 You are all set!
