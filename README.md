## 🛠️ Installation Steps
###  Create React-App.
  1. Create React-app

```bash
  npx create-react-app my-app
```

  2. Change the working directory

```bash
  cd my-app
```
  3. Run the app

```bash
  npm start
```

🌟 You are all set!
